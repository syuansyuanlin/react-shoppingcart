# 歡迎我的購物網站

### 前端指令

前端資料夾位置：`<path>/<project name>/frontend`\
安裝程式庫：`npm install`\
啟動：`npm start`

### 啟動後端

後端資料夾位置：`<path>/<project name>/backend`\
安裝程式庫：`npm install`\
啟動：`node index.js`
